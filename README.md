# \# 💬 Real‑Time Chat Application

# 

# A lightweight, real‑time chat application built with \*\*Node.js, Express, and Socket.IO\*\*. Multiple users can join, send messages instantly, and see who's online – no page refresh, no complex setup.

# \## ✨ Features

# 

# \- 💬 \*\*Real‑time messaging\*\* – messages appear instantly on all connected clients

# \- 👥 \*\*Multiple users\*\* – unlimited participants, each with a custom username

# \- 🟢 \*\*Online presence\*\* – see when someone joins or leaves the chat

# \- 🎨 \*\*Clean, responsive UI\*\* – works on desktop and mobile

# \- 🚀 \*\*No database required\*\* – in‑memory storage for demo purposes (easy to extend)
